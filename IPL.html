This project combines the thrill of live IPL cricket with the analytical power of Power BI, integrating real-time data from Rapid API’s Cricbuzz Cricket to create a fully dynamic and interactive dashboard. The 3-page design includes:

  • Intro Page: A sleek landing page with a “Dashboard” button leading to live visualizations.
  • Live Data Visualization: Real-time match stats and updates, refreshed automatically.
  • Points Table: Interactive standings with detailed match statistics.
